① 为什么这个 App 需要 Azure？

很多人会问：为什么不用国内的语音服务？这是因为德语发音评估是一个非常专业的功能，目前国内平台（讯飞、阿里云、腾讯云等）均不支持德语，只有中文和英文。

🎯
发音评估（核心功能）
你跟读后，系统分析你每个词的发音准确度，给出 0-100 分，标出哪个词需要改进。这需要 Azure 的 Pronunciation Assessment 技术，国内无替代品。
🔊
德语标准发音播放（TTS）
播放发音时用的是 Azure 的德国母语 Neural 声音（Katja、Conrad 等），语调自然，是真正的德语母语发音，不是机器人声。
🌍
为什么是微软 Azure？
微软在语音技术上深耕多年，德语是其重点支持语言之一。Azure Speech 是目前对非英语发音评估最准确的商业 API，连 Duolingo 等学习 App 背后也在用类似技术。

② 费用说明

Azure Speech 服务有永久免费层（F0），不是试用，是长期有效的免费额度：

功能 每月免费额度 够用吗？
德语 TTS（播放发音） 免费 50万字符 可播放约 2 万句话，完全够
发音评估（跟读打分） 免费 5 小时音频 每天练习 10 分钟可用 30 天

✅ 关于信用卡
注册时需要绑定 Visa 或 Mastercard 信用卡，但这只是身份验证用途。只要你设置消费上限为 $0（下方有教程），账单绝对不会产生任何费用。Azure 明确承诺免费额度内不扣费。
⚠️ 银联卡无法使用
目前 Azure 国际版只接受 Visa 或 Mastercard，不接受银联卡。如果你没有这两种卡，可以考虑办一张招商银行或工商银行的双币信用卡。
③ 申请步骤（约 10 分钟）

第一步：注册 Azure 账号

1
打开 Azure 官网
访问 portal.azure.com，点击"免费开始"或"Try Azure for free"
2
选择 "Azure free account"
右侧选项，点击 "Try Azure for free"，有 $200 免费额度（30天）+ 永久免费服务
3
注册微软账号
用邮箱注册或登录已有微软账号（Outlook、Hotmail 均可）
4
填写个人信息和信用卡
填写姓名、地址，绑定 Visa/Mastercard。系统会验证扣 $1 然后退回，这是正常流程

第二步：创建语音服务

1
进入 Azure Portal
登录后在顶部搜索栏输入 Speech，选择 "Speech services"
2
创建新资源
点击"创建"，填写名称（随意，如 my-speech），区域选 East Asia（香港，延迟低）
3
选择免费层
Pricing tier 一定选 Free F0，这是永久免费的层级
4
点击创建，等待约 1 分钟
创建完成后进入资源，左侧菜单点 "Keys and Endpoint"，复制 Key 1 和 Location/Region（如 eastasia）

第三步：设置消费上限（强烈建议）

1
进入 Cost Management
在 Azure Portal 左侧菜单找 "Cost Management + Billing" → "Budgets"
2
设置预算上限为 $0
创建一个 Budget，金额设为 $1，超出时发邮件提醒。这样万一超出免费额度你会立刻知道
④ 填入 App
1
打开 App，点击顶部"API 设置"
展开设置面板，可以看到两个输入框
2
填入 Azure Speech Key
粘贴从 Azure 复制的 Key 1（32位字符串）
3
确认 Region
如果创建时选了 East Asia，填 eastasia；其他区域填对应名称
4
点击"保存"，状态变为绿色"✓ 已配置"即成功
Key 保存在你自己的浏览器本地，不会上传到任何服务器
🔐 你的 Key 安全吗？
完全安全。Key 只存储在你自己的设备浏览器里（localStorage），不经过任何中间服务器。只有你的设备能看到它，就像密码保存在浏览器里一样。
⑤ 常见问题
🇨🇳 国内网络能访问 Azure 吗？
注册页面（azure.microsoft.com）在国内部分地区可以直接访问，部分地区需要翻墙。注册完成后，App 调用 API 时走的是 East Asia（香港）节点，国内网络一般可以直接使用，延迟约 50-100ms，不影响体验。
💳 绑卡后真的不会扣钱吗？
注册时系统会验证扣 $1 然后立即退回，这是正常流程。只要使用 Free F0 层级，且在免费额度内，不会产生任何费用。建议同时设置消费预警，双重保险。
📊 如何查看我用了多少额度？
登录 Azure Portal，进入你的语音服务资源，左侧菜单有 "Metrics" 可以查看使用量。也可以在 "Cost Management" 里查看费用情况。
🔄 Key 过期了怎么办？
Azure Key 本身不会自动过期，除非你手动"重新生成"。如果 App 突然提示出错，可以去 Azure Portal 的 "Keys and Endpoint" 重新复制一次 Key 填入即可。
🗑️ 不想用了怎么删除？
在 Azure Portal 找到你的语音服务资源，点击"删除"即可。删除后不再产生任何费用，Key 也立即失效。
📱 手机上能用吗？
可以。把 App 网址在手机浏览器打开，iOS 建议用 Safari，Android 建议用 Chrome。录音和发音评估功能在手机上同样有效。还可以把网址添加到手机主屏幕，像原生 App 一样使用。
